star-rpg-framework
==================

JS-based RPG framework based on cocos2d familiy

run sample app
==================
```sh
$ cd star-rpg-framework
$ (cd html; python -m SimpleHTTPServer)
$ make bake
```

http://localhost:8000/

setup typescript environment
==================
npm install -g typescript
