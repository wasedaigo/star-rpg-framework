star-rpg-framework
==================

JS-based RPG framework based on cocos2d familiy

run sample app
==================
```sh
$ cd star-rpg-framework
$ node script/server.js
$ ./script/build.sh
```

http://localhost:8000/

setup typescript environment
==================
npm install -g typescript
