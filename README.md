star-rpg-framework
==================

JS-based RPG framework based on cocos2d familiy

run sample app
==============
```sh
$ cd star-rpg-framework
$ ./script/server.js
$ ./script/build.js
```

http://localhost:8000/

setup typescript environment
============================
npm install -g typescript

editor plugin
=============

how to build
============
